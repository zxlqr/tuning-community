# SQL Миграции

Эта папка предназначена для хранения SQL скриптов миграций, которые выполняются вручную.

## Когда использовать

Используйте эту папку для:
- Сложных миграций, которые проще написать на SQL
- Миграций данных (data migrations)
- Оптимизации индексов
- Других SQL операций, которые не покрываются Django migrations

## Django Migrations

Обычные миграции Django находятся в соответствующих приложениях:
- `backend/accounts/migrations/`
- `backend/shop/migrations/`
- и т.д.

## Формат именования

Используйте формат: `YYYY-MM-DD_description.sql`

Пример: `2025-12-02_add_indexes.sql`
